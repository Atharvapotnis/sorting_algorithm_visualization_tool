# Sorting Algorithm Visualization Tool

This repository contains an interactive visualization tool for common sorting algorithms, designed to help understand how different algorithms work step by step.

## Live Demo

You can view and use the webpage here:  
https://atharvapotnis.github.io/sorting_algorithm_visualization_tool/

## Features

- Visual comparison of sorting algorithms
- Step-by-step animation of swaps and comparisons
- Simple and intuitive interface for learning and teaching

## Usage

Open the live demo link above in your browser.  
No installation or setup is required.
